Apple은 ‌iCloud‌ 키 체인 암호를 PC에서 사용할 수있는 Windows 용 Google Chrome 브라우저 용 iCloud 암호 확장 프로그램을 출시했습니다.
 

![icloud passwords chome extension](https://images.macrumors.com/t/iSZS5DirDH6F6J1mF3sIVVRx5gA=/2500x0/filters:no_upscale():quality(90)/article-new/2021/02/icloud-passwords-chome-extension.jpg)

간단히 "‌iCloud‌ 암호"라고하는이 확장 프로그램은 사용자가 Apple의 Safari 브라우저에서 생성 된 Google Chrome의 암호에 액세스 할 수 있도록합니다.
 또한 Chrome에서 생성 된 암호의 ‌iCloud‌ 동기화를 활성화하여 Apple 장치에서도 사용할 수 있습니다.
 

이 기능은 지난주 Apple이 확장 프로그램을 사용할 수 없었음에도 "iCloud‌ Passwords Chrome Extension 지원"이 포함 된 새로운 버전의 Windows 10 용 iCloud를 출시했을 때 암시되었습니다.
 

‌iCloud‌ 암호는 이제 Windows 및 Mac 용 Chrome 웹 스토어에서 사용할 수 있습니다.
